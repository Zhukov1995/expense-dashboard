# КэшФлоу — трекер расходов

Демо-проект для портфолио. Данные демонстрационные.

Дашборд личных финансов: KPI, графики по категориям и месяцам, добавление,
редактирование, удаление и фильтрация транзакций, экспорт в CSV. Светлая, тёмная
и системная темы.

## Стек

Vite, React, TypeScript, Tailwind CSS, Zustand (persist), Recharts, Vitest.

## Запуск

```bash
npm install
npm run dev
```

## Сборка и тесты

```bash
npm run build
npm test
```
